Automated endpoint validation using application dark theme design tokens

1. Register User

cURL Request

curl -s -X POST http://localhost:3000/api/v2/auth/register -H 'Content-Type: application/json' -d '{"name":"User 48494","email":"user48494@example.com","password":"password123","accountNumber":"20624262128492"}'

JSON Response

{
  "message": "User registered successfully."
}

2. Login User

cURL Request

  "message": "Logged in successfully",
